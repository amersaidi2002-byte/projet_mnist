# projet_mnist
Francais:
Projet d’IA embarquée visant à entraîner et valider des réseaux de neurones (CNN et MLP) sur la base de données MNIST ainsi que sur un jeu de données personnalisé. Les modèles sont ensuite implémentés en C from scratch et déployés sur une carte Raspberry Pi afin de réaliser de la détection en temps réel à partir d’une caméra.

English:
Embedded AI project focused on training and validating neural networks (CNN and MLP) on the MNIST dataset as well as a custom dataset. The trained models are implemented from scratch in C and deployed on a Raspberry Pi to perform real-time detection using a camera.
